# Task Management Web Application
A web-based task management system built using Flask and SQLAlchemy.  
This application allows users to register, login, and manage their daily tasks efficiently.

---
## 🚀 Features
- User Registration and Login (Authentication system)
- Add, Update, Delete tasks (CRUD operations)
- Store tasks using database (SQLite)
- Organized backend using Flask app factory pattern
- Clean UI with HTML and CSS
---

## 🛠️ Tech Stack
- Python
- Flask
- SQLAlchemy
- HTML, CSS
- SQLite

## ▶️ How to Run the Project

1. Clone the repository:
git clone https://github.com/sujay-vyas/Flask-Projects.git

2. Navigate to the folder:
cd Flask-Projects

3. Create virtual environment:
python -m venv venv

4. Activate virtual environment:
venv\Scripts\activate

5. Install dependencies:
pip install flask sqlalchemy

6. Run the application:
python run.py
